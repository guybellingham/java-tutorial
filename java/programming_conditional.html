Conditional statements.

 looping
The if statement conditionally executes blocks of code (between curly braces). The if keyword must be followed by an expression in parentheses that evaluates to a boolean (true / false) value. Note that unlike some other languages, the Java if statement does not accept a numeric value of 0 (to mean false) or non-zero (to mean true). 
if (x > y) { 
  //statement block only executed when x > y is true 
} 
if ((x > y) && (x > z)) { 
  //statement block executed when x > y AND x > z is true 
} else {
  //statement block executed if condition is false 
} 
//nested if statement 
if (day.equals("Saturday")) {
  //statement block executed when day String contains "Saturday" 
} else if (day.equals("Sunday")) {
  //statement block executed when day String contains "Sunday" 
} else {
  //statement block executed when day String is something else 
}

The switch statement allows conditional execution, based upon a number of cases. If a case is true, then its corresponding statement/s are executed. Note that the switch keyword is followed by a data type of byte, short, char or int in parentheses (you cannot use it to test object variables or decimals). Curly braces enclose any number of case statements, which execute Java statements depending upon the value of the argument given to the switch. The comparand in a case statement, must be a constant or literal value, no expressions are allowed.

int dayOfWeek = myCalendar.get(Calendar.DAY_OF_WEEK);
switch (dayOfWeek) {
  case 1:
   System.out.println(“Monday - football”);
   break;
  case 2:
   System.out.println(“Tuesday - garbage day”);
   break;
  case 3:
   System.out.println(“Wednesday - pickup pizza”);
   break;
  case 4:
   System.out.println(“Thursday”);
   break;
  case 5:
   System.out.println(“Thank God it's Friday”);
   break;
  default:
   System.out.println(“Its the weekend!”);
   break; 
}
Remember to include a break, throw or return statement after the statement/s that belong to each case, otherwise the following case statements are also evaluated. A 'catch all' default case can be specified anywhere within the switch statement.
The last type of conditional statement is the ternary (three way) operator.
