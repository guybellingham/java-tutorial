java.awt.Panel

 Window

A Panel serves as a basic Container within which you 'nest' other containers and components. Usually an Applet or Frame serves as the top level container. By adding groups of components to a panel, and then adding the panel to the top level container you can combine layout strategies to get the spatial organization you need.

The default LayoutManager for a panel, is the FlowLayout.
