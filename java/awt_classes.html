The AWT classes

 Component

The java.awt package contains the main toolkit classes and interfaces, chief amongst which are the Component and Container classes. The class diagram below shows that all the classes for graphical controls inherit from the Component class.

figure: The AWT class hierarchy.

 

There is a relationship between the Component and Container classes, that says 'a container may contain zero to many components' and that 'a component may be contained by a container'. In this way you can add GUI Components to Containers to build up your required interface. Notice that you can nest one or more containers within another container. The process of composing a complex user interface, by nesting components and containers within one another, will become very familiar to you as you use the AWT.

The Component class implements the ImageObserver, MenuContainer and Serializable interfaces. In addition the Scrollbar and ScrollPane classes implement the methods of the Adjustable interface, and List, Choice and Checkbox classes implement the method of the ItemSelectable interface.

Not shown here are the MenuComponent class and its descendents, MenuBar and MenuItem etc.
