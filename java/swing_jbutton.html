javax.swing.JButton

 JList

Swing JButtons generate ActionEvents when clicked.

Because they are descended from JComponent, they share the abilities of all Swing components to have borders, keyboard accelerators, pluggable look-and-feel and accessibility features. Much of the functionality common to all buttons is defined in the javax.swing.AbstractButton class, which has methods to get and set the text and icons of the button, and control alignments.

 figure - JButton has a model and a user-interface.

 

Notice that in Swing, a JMenuItem is a type of button. Also the JCheckBox and JRadioButton inherit from the JToggleButton!

You can group buttons by adding them to an instance of the ButtonGroup class.

ButtonGroup bg = new ButtonGroup( );
bg.add(radiobutton1);
bg.add(radiobutton2);

It's best to add buttons to a container (like a JPanel) that has a GridLayout or the new BoxLayout so that they have similar sizes.
