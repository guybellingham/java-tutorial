Applet - circle slider control.

 Advantages of Objects

This Java Applet, is made up of several Objects.

This object inherits from the java.applet.Applet object type. The java.applet.Applet object type is provided with the Java runtime, and defines small programs that can be embeded in a web page and run inside a web browser.
This particular Applet contains another object called CircleSlider written by me, which inherits from the java.awt.Canvas object type. It also contains an event handling object, and properties I have called 'theta' and 'rho', (displayed at the top of the Applet).

  • The event handler in this Applet is another object, and it responds to mouse drag input and the arrow keys.
  • If you right-click on the Applet it displays yet another object, a Popup-menu, through which you can change the properties of the underlying Canvas object.
