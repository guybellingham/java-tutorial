java.awt.Label

 List
A Label, serves to contain text and 'label' other visual controls. Like all AWT Components, you can get and set the text , color and font of a Label object.
This Applet contains a TextField at the top, a Label in the middle. Buttons along the bottom control the Font used by the Label. The textfield passes letters and digits to the label as you type. Hit the <Font> button to cycle through the available fonts. Hit the + and - button to increase and decrease the font size. Hit the clear button to reset.
