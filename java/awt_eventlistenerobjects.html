Listening for Events

 Fonts
  This Applet, uses all four strategies to handle AWT Events.
1. The Applet itself implements MouseMotionListener, and MouseListener interfaces and all the methods therein. The Applet adds itself to the white drawing panel to listen for Mouse Events.
2. A standard adapter class implements ActionListener, and is added to the button to listen for an ActionEvent and invoke the clear( ) method in the Applet class.
3. An inner class listens for ItemEvents from the drop down Choice box.
4. Anonynous classes that implement the ItemListener interface, are added to the 2 Checkbox controls (Outlines vs Filled).
