Applet - drawing shape objects.

 Objects

This Java Applet, contains a drawing 'Canvas' object, that stores up to 64 Shape objects.

You can choose the types of Shapes to draw and set color and fill properties of the shapes, using the drop downs and radio buttons at the top.

  • Click and drag to create new Shape objects.
  • SHIFT + drag to move objects around.
  • CTRL + drag to change the shapes.
  • Double click to clear the canvas.

Important Note: This site contains a number of interactive Applets to demonstrate important concepts. Your computer may need the current version of Sun MicroSystem's browser plugin available for free download from Suns website.
