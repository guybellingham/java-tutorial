Example Swing JApplet

 Swing

Below is an example JApplet drawing program that makes use of a JMenuBar.
Click and drag to draw shapes, double click to clear the canvas. SHIFT + drag to move shapes around. CTRL + drag to change an existing shape object.

The JMenus and JButtons have been assigned Mnemonics. For example hit ALT + C, to open the color picker, hit ALT + S to select the Shape menu.
Note: You may need to click within the applet, to give it input 'focus'.

Note: The JMenuItems under the JMenus have been assigned KeyStroke accelerators.
For example, to draw Rectangles hit CTRL + R, to draw Ovals hit CTRL + O, to draw filled hit CTRL + F, and hit CTRL + Z to undo the last shape.

To change the SWING user interface open the Look & Feel Menu.

The newest web browsers support Applet printing to your local printer, but only if you OK it. To print the canvas, hit CTRL + P. If nothing happens, your browser probably does not support applet printing.
Note: If you have enabled the Java console in your browser, select View > Java Console to find out what the problem is.
