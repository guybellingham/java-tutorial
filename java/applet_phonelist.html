Java contents. Phonelist Applet    
This Applet is signed by gus, and both reads and writes files to your local file system.
If you want to be able to File > Save a phone list, you will need to grant it a FilePermission to write. Here is an example phonelist file, you can download and open in this applet.

Use the policytool to modify the policy file given on the trusted applet page.
Here is an example of how the Applet looks, once you get it running.

Ctrl+F to Find a phonelist entry.
Ctrl+T to Sort the list.
Ctrl+O to open a phonelist file.
Ctrl+S to save a phonelist.
Dbl-Click an entry to retrieve it for updating.
Ctrl+R to remove a phonelist entry.
This the File > Open Dialog. It is a custom component written by me, using a JTree and a custom TreeModel.
