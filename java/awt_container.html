java.awt.Container

 Dialog

The Container class extends the abstract Component class. Containers can therefore behave just like Components, and a Container object can 'contain' other Container objects.
A Containers main job is to keep track of all the components it contains.
The job of arranging and sizing all the components in a container, is contracted out to a separate LayoutManager object.
