java.awt.Scrollbar

 TextField

Scrollbars, can be oriented either horizontally or vertically. They are often used to scroll text areas, but can be used to select a numeric value from a range of possibilities, from a minimum to a maximum integer value.

 

 

Here are some of the more useful Scrollbar methods.

int getValue()
int getMaximum()
int getMinimum()
void setBlockIncrement(int x)
void setLineIncrement(int x)
void setOrientation(int x)

Try dragging and clicking beside the Scrollbars in the Applet at left.

You'll see that they generate an AdjustmentEvent with a number of specialized ids, depending on how you use them.

Dragging the box to a new location generates Events with an id equal to AdjustmentEvent.TRACK.
While clicking the arrows at either end scroll the bar up or down a line at a time, see AdjustmentEvent.UNIT_INCREMENT and UNIT_DECREMENT.
If you click ahead or behind the box, the bar scrolls a block at a time. see AdjustmentEvent.BLOCK_INCREMENT and BLOCK_DECREMENT.

The maximum, & minimum values are defined when a new scrollbar is constructed, as is the orientation.

However you can get and set the size of the Line or Page increment in a Scrollbar at any time.
