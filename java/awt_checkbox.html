java.awt.Checkbox

 Choice

AWT Checkboxes have a split personality.

When added to a container they look like the square boxes on the right. However they can also belong to a CheckboxGroup, in which case, they look like the circular 'radio buttons' on the left.
You use the method setCheckboxGroup(CheckboxGroup) to group Checkboxes.

Notice that only one 'radio button' in a CheckboxGroup can be selected at any one time.
Also a radio button does not check and clear when repeatedly clicked like the square check boxes.
