java.awt.TextField

 TextArea

A TextField, generates an ActionEvent, only when the ENTER key is hit, while the cursor is focused upon it.
When in editable mode, you can click in the text field, and 'enter' the details required. Here are some of the more useful methods of TextField:

int getColumns();
void setEchoChar(char c);

Note: The Java version 1.0 versions of the AWT, do not support keyboard tabbing between components! The Java 1.1 'Swing' components fix this failing.

 A TextField is a single line text entry box, that usually requires some label in front of it to explain what the user needs to enter there.
It inherits several methods from the TextComponent, class which deal with inserting, selecting and editing text.

Click in each of the TextFields in the Applet at right..

Type invalid and valid SSN and ZIP values and hit the ENTER key to see what happens.

The first two fields are subclasses of the TextField component, that can take a Validator object when they are constructed.

  • The SSN field has an SSNValidator object in the background which checks the format of the string you enter into the field.
  • The ZIP field has a USZIPValidator object, behind the scenes which checks the format of the zip code when you hit the Enter key.
  • The password field is a regular TextField with the echo character set to '*'.
