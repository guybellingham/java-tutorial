Java Arrays.

 Dates

An Array is a way of defining space for an internal table of data or object references. Note that only references to objects are stored in an array and not the objects themselves.
There are 3 steps in creating an array:

  1. Declare the array variable. You start out by declaring what type of data or object the array can contain.

float myFloatTable[ ];  //datatype table
String [ ] myTextTable; //contains references to String objects
java.awt.Point [ ] [ ] myCoordinatesTable; //two dimensional table

Notice that square brackets are used to denote an array, and they can precede or follow the array variable name. At this stage all we have is a 'handle' to an array. In the last example the array has two dimensions, and consists of references to Java Point objects.

  2. Allocate space for the new array. An array can be fixed or variable length (x must be a whole number). However once space is allocated for an array it can only hold a determinate number of elements. Attempts to access an array beyond its bounds will lead to a runtime exception (java.lang.ArrayIndexOutOfBoundsException).

myFloatTable = new float[6];
myTextTable = new String[x];
myCoordinatesTable = new java.awt.Point[2][2];

Note: All Arrays have a data variable called length, which you can inspect to find out how many elements can fit into the array. For multi-dimensional arrays, each dimension has a length property. This and the fact that you must construct an array with the keyword new, should give you a clue that arrays are actually objects in their own right.

  3. Assign values to the entries in the new array. Java automatically initializes arrays of numerics to zero, chars to the unicode null character, and boolean arrays to false. An array of object references will have null pointer values until you assign objects to the elements of the array. Java provides a quick way to initialize an array at the point the new array is created.

//all three steps in one!
float [ ] myFloatTable = {1.2F, 3.0F, 3.5F, -4.25F, 5, 6.18F}; 
String [ ] myTextTable = {"zero","one","two","three"};
//simple 2D table initialization!
int [ ][ ] myNumberTable = { {0,1,2}, {3,5,8}, {13,21,34} };

//complex 2D tables can use nested loops to init elements
for (int i=0; i < myCoordinatesTable.length; i++) { 
  for (int j=0; j < myCoordinatesTable[i].length; j++) { 
    myCoordinatesTable [i][j] = new Point(i*2,j*2);
  }
}
Note: Items in an Array start at an index position of zero, and extend upto 1 less than the number of rows or columns in the array.
The java.util.Arrays class, has some useful class (static) methods for comparing, sorting and filling arrays of datatypes and objects.

The java.lang.System class, has a class (static) method for copying elements between arrays. The method shown below copies the given number of array elements (length) from the specified source array (src), beginning at the specified position (src_position), to the specified position (dst_position) of the destination array (dst).

public static native void arraycopy(Object src, int src_position, Object dst, int dst_position, int length) {
}

An IndexOutOfBoundsException may be thrown if copying would cause access of data outside the bounds of the arrays. An ArrayStoreException may be thrown, if an element in the source array is not assignment compatible with the destination array.
