java.awt.Button

 Canvas
When enabled and showing on screen, clicking a Button with the mouse generates an 'Action' event, which your applet can detect and 'act' upon.

About the only methods you will need to call in a Button are the setEnabled() and setLabel() methods.

Note: when a button is disabled or hidden, the mouse clicks no longer generate events.
The Applet at left, lists all the events that are fired by the AWT when you click the button.
