javax.swing.JLabel

 JButton

javax.swing.JLabel is just about the simplest Swing component. Yet the Swing classes provide a lot more funcionality that the corresponding GUI component in the AWT .

Swing provides a much richer set of constructors and methods in the JLabel class, than are found in the AWT Label class. Like the java.awt.Label, you can set the Font and Color of a JLabel.
However since JLabel extends JComponent, JLabels also support borders, image icons, and keyboard accelerators.

Swing also provides a package called javax.swing.borders that contains various types of Border classes. You can also use the javax.swing.BorderFactory class which has static methods you can call, to 'manufacture' the most common types of borders. You can create various styles of border around swing components with code similar to that shown below, which loads an image and adds it to a JLabel as an icon and then surrounds the label with a raised bevel border :

Image image1 = getImage(getCodeBase(), "images/blue_ribbon.gif");
ImageIcon icon1 = new ImageIcon(image1);
JLabel label1 = new JLabel("Label1", icon1, SwingConstants.CENTER);
/* It's simpler to use the BorderFactory to do the following
   BevelBorder border1 = new BevelBorder(BevelBorder.RAISED);
   label1.setBorder(border1);
*/
label1.setBorder(BorderFactory.createRaisedBevelBorder());
