You cannot edit the text in this TextArea, however you can select text with the mouse.

Applets are restricted from reading or writing to the file system of the computer on which they are running, so there is no need to worry about a malicious Java Applet infecting your system from the Internet.

However you should only download executable files from trusted sources.
