java.awt.Frame

 Panel

 The Frame component, is not controlled by a LayoutManager, and you need to call the

setSize(int width, int height)

method to give a Frame, sensible default dimension before making it visible.

A Frame has a BorderLayoutManager by default.

The button builds and shows a Frame that draws an Image with a blue dot. The Frame also has a MenuBar with a Help Menu on it.
If you select the "About" MenuItem, a Dialog box is opened.

Note: Most browsers mark Frames and Dialogs that are opened by an Applet, with a warning at the bottom. This is just to let you know that this is not a regular window, and is owned by the Applet downloaded from my web site.
