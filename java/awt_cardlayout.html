java.awt.CardLayout

 AWT Events
This LayoutManager uses the 'deck of cards' metaphor and ignores the preferred size of any component added, and stretches it to fit the entire container. Very often you 'nest' other containers within a container that has this LayoutManager.

There are 2 constructors :

CardLayout( );
CardLayout(int hgap,int vgap);

Note: you will need a browser that supports Java 1.1 to make this applet work properly. Some versions of IE 4.* have bugs in their event handling support, you should probably upgrade to the latest version.
Note: The CardLayout has a special methods , which display the components in the container's Component[ ] array one at a time. 
next(Container c); // displays the next component in c
previous(Container c); // displays the prev component 
first(Container c); // displays the first component added to c
last(Container c); // displays the next component

You can also go directly to a particular 'card' in the stack as long as you have named it using a String when you add( ) it to the container :

show(Container c, String name); // displays the named component
