java.awt.Dialog

 Frame
Dialog boxes are really designed for use in 'free standing' Java Applications, that have a Frame of their own.

Note: Applets aren't designed to make use of Dialogs. The Applet below has to cheat to show you a Dialog. If you don't see the Dialog box, make sure you hit the button forcefully, and that it then becomes disabled (grayed out). You may need to use Alt + Tab to switch to the Dialog box, which may appear in the corner of the screen.

Note: Some 'older' browsers do not support an Applet raising a custom Event, and will throw a SecurityException. If the Dialog does not appear or will not accept a password you type, open the View menu, and select Java Console, to find out what is going on. 

 When you hit the Button in the Applet at left, it displays a 'modal' Dialog box, with a TextField for you to enter a password into. You may hit the Cancel button to dismiss the dialog, but no password will register in the Applet on this page. If you do enter a password, it will be shown here.

A modal dialog box, grabs all the input to a Window so that you must dismiss it before you can continue.
